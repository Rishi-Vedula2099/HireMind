# HireMind

A fully personalized AI interviewer that simulates real hiring pipelines based on job role, company, and candidate resume.

## Features

- **Personalized Interviews**: Tailored questions based on specific job roles and company cultures.
- **Resume Analysis**: Simulations that take into account the candidate's actual background.
- **Real-time Feedback**: Detailed analysis of interview performance.

## Getting Started

1. Clone the repository.
2. Install dependencies (coming soon).
3. Set up environment variables.
4. Run the development server.
